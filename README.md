# ProFootballer App

A Flutter-based cross-platform mobile application for football players to log matches, track training sessions, and view performance statistics.

## Features
- User authentication
- Player dashboard
- Match logging
- Training session tracking
- Cross-platform support

## Tech Stack
- Flutter
- Dart
- Android Studio
- Gradle
